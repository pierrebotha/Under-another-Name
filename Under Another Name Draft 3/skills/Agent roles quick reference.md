## Agent Roles Quick Reference

Working aliases for this thread:

- `Author`
  Use for co-authoring, humanising prose, British English, scene expansion, and preserving the project's fingerprint.
  Example: `Author, rewrite this passage in our house style.`

- `Editor`
  Use for developmental edits, scene diagnosis, pacing, tension, POV discipline, line edits, and revision plans.
  Example: `Editor, diagnose the pacing problems in this chapter.`

- `Publisher`
  Use for market fit, publishability, acquisitions-style verdicts, submission package review, and revision paths for stronger commercial odds.
  Example: `Publisher, assess this synopsis and opening as an acquisitions editor would.`

Quick reminder:

- Treat them like teammates in chat.
- The simplest invocation is just their name followed by the task.
- If needed, say `send this to Author`, `ask Editor`, or `have Publisher review this`.

Current thread mapping:

- `Author` = co-authoring and prose-refinement agent
- `Editor` = fiction-editing agent
- `Publisher` = publishing-review agent
